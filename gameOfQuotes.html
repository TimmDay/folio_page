This web dev project was designed to demonstrate the implementation of data from a free API as well as from a hard coded JSON data structure. It shows quotes from G.R.R. Martin's Game of Thrones, (on the left, hard coded) and random famous quotes from esteemed real life people (on the right, from the forismatic API).
Javascript was applied to the author names, to link to appropriate wiki articles for more information.
